# Solution to algorithm questions from Sir Mark.

## Language

Python 3

## Installing

```bash
git clone https://github.com/dodziraynard/sir-mark-algo.git
cd .\sir-mark-algo
pip install -r .\requirements.txt
```

## Running

### Power calculator

```bash
cd .\sir-mark-algo
python .\power.py
```

### Testing Power calculator

````bash
cd .\sir-mark-algo
python py.test

### Text printing

```bash
cd .\sir-mark-algo
python .\print_text.py
````

### Number of languages spoken

```bash
cd .\sir-mark-algo
python .\languages.py
```
